# Hotel_Bookings_Analysis
The Hotel Booking Analysis Dashboard is designed to analyze booking patterns, customer behavior, and revenue performance for a hotel business.
This project provides insights into cancellation trends, revenue generation, guest preferences, and seasonal booking patterns using Power BI for interactive data visualization and decision-making.

#Dataset Information
The dataset includes detailed booking records with attributes such as:
   1.Hotel Type (City Hotel or Resort Hotel)
   2.Lead Time (days between booking and arrival)
   3.Arrival Date (year, month, week number)
   4.Stays in Week Nights and Weekend Nights
   5.Adults, Children, Babies
   6.Meal Type and Market Segment
   7.Country
   8.Customer Type (Transient, Contract, Group, etc.)
   9.Deposit Type and Reservation Status
   10.ADR (Average Daily Rate)
Data was cleaned and transformed in Power BI for accuracy and visualization purposes.
